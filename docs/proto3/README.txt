iPad app installation instructions:

Open the project in xCode. Then go to accounts in preferences and add the developer account to the apple ids list. After that connect the ipad and the organizer window should pop up automatically. If the window doesnâ€™t pop up, go to the window menu and open it by clicking on organizer. Then select the ipad and click use for development. Next you can close the organizer window. After that you can select the ipad as the target in the top left corner of the main window. Finally, click the run button and the app will be downloaded to the ipad.
